Note: 1. Server filen kan man inte kolla på eftersom att den innehåller känslig kod såsom: API nyckel till AI. 2. Miniräknaren fungerar bättre när servern är på (altså AI + servern är ingång)

För att få igång servern skriv in dessa kommando i två bash terminaler en för server och en för client (kan hittas i visual studio code) För kommadot för att starta servern: 
cd server 
och
node index.js
I bash server terminalen

För att starta client skriv in kommandot:
cd client
serve

NOTE: borde finnas med någon slags tutorial eller video med i filen för hur man startar programmet, AI'n kommer INTE fungera om inte servern och client är igång.
